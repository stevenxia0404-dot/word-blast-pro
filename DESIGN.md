# Word Blast Pro - 完整设计文案

## 项目定位
面向小学2年级（7-8岁）的中英单词学习 PWA 游戏。通过配对消消乐 + 拼写模式 + 完形填空，训练认读→默写→运用能力。

## 技术栈
- React 19 + Vite 8 + TypeScript 6
- Tailwind CSS 4
- vite-plugin-pwa（离线可用、可安装）
- Canvas 渲染引擎（游戏画布）
- Web Audio API（音效合成）+ SpeechSynthesis（TTS 英文发音）
- 字体：Fredoka One（标题）+ Quicksand（正文）+ PingFang SC（中文回退）

## 视觉风格
- 背景：柠檬绿→青柠渐变 `linear-gradient(135deg, #fef08a 0%, #d9f99d 100%)`
- 卡片配色：英文=柔和蓝(#eff6ff/#bfdbfe/#1e40af)、中文=温暖橙(#fff7ed/#fed7aa/#9a3412)、选中=生机绿(#84cc16)、匹配成功=绿(#22c55e)
- 圆角大卡片 `rounded-[2.5rem]`，阴影层次丰富
- 粒子礼花效果+弹性重力动画+抖动反馈
- `.bubbly-font` 用于标题和数字

## 词库结构（来自 U1-U9词汇.md）
- 基础单词 #1-90：单字，如 robot↔机器人、music↔音乐
- 核心短语 #91-102：短词组，如 feed the cows↔喂奶牛、see a film↔看电影
- 完整句子 #103-108：整句，如 My mum is a great cook.↔我的妈妈是一位超棒的厨师。

## 线性通关链路
不按「模式」手动切换，按「关卡」线性推进。每关 3 个小节闭环：认→拼→用。
```
关卡 1：动物/人物单词（选取约10个词）
  ├─ 第1小节 配对消消  3×4网格（6对）  认读
  ├─ 第2小节 字母拼写  看中文→点字母拼英文  默写
  └─ 第3小节 完形填空  英文句子里挖空，看中文填  运用
  ✅ 通关 → 解锁关卡2

关卡 2：植物/自然单词
  ├─ ...
  ...
关卡 N：短语（2×3网格，3对，卡片变大）
  ├─ 第1小节 配对消消（短语版）
  ├─ 第2小节 选词拼短语（看中文→按序点英文单词）
  └─ 第3小节 短语内单词拼写
  ✅ 通关 → 解锁下一关

关卡 N+1：句子（2×2网格，2对，卡片最大）
  ├─ 第1小节 配对消消（句子版）
  ├─ 第2小节 完形填空（整句挖空）
  └─ 第3小节 连词成句（看中文→按序拼出完整英文句）
  ✅ 最终通关 → 证书
```

## 两种游戏模式（小节级别复用）

### 配对模式（已有，保持）
- 网格中随机分布中英文卡片
- 点击一张选中，再点另一张：pairId 相同且类型不同 → 配对成功 → 爆炸+语音
- 配对成功后重力下落补新卡片，防重复算法确保容器内无重复配对
- 配错→卡片抖动→Combo 断掉

### 拼写模式（新增）
- 顶部：中文提示 + 对应 emoji
- 中间：字母/单词槽位（根据答案长度动态生成）
- 底部：候选字母/单词块（含干扰项，打乱顺序）
- 点击候选块→填入第一个空槽，填满自动判定
- 全对→礼花+TTS朗读
- 点错→槽位不填入，候选块抖动

## 积分体系

### 赚分
```
配对正确     +100
拼写正确     +100
完形填空正确  +100
Combo 额外   +50 × 当前连击数
```

### 花分（求助机制，同一道题连续错2次后触发）
```
求助等级      消耗积分    效果
────────────────────────────────────
L1 首字母     300分      显示第一个字母（A _ _ _ _）
L2 揭露一半    500分      显示前一半字母（A p p _ _）
L3 直接跳过    800分      揭晓答案，本题跳过
```

### 求助后得分
```
求助等级      正常      求助后
──────────────────────────────
L1 首字母     +100      +30
L2 揭露一半   +100      +10
L3 直接跳过   +100       0
```

### 求助规则
- 同一道题连续错2次才出现求助按钮（非强制）
- 中间对了一次→错误计数清零
- L1→L2→L3 必须顺序升级
- 使用求助→本题不计 Combo
- 求助不计入准确率（算跳过）
- 所有模式同理
  - 配对模式：同一对配对连续错2次
  - 拼写模式：同一单词拼写连续错2次
  - 完形填空：同一填空连续错2次

## 星级评定（每关）
```
3星 ⭐⭐⭐：准确率 ≥ 90%
2星 ⭐⭐  ：准确率 ≥ 70%
1星 ⭐   ：通过即可
```
累计星星用于解锁后续大关卡。

## 通关证书

### 触发条件
全部关卡通关后弹出

### 视觉
- 背景：白底 + 绿黄渐变圆斑装饰
- 主体：白色圆角大卡片 `rounded-[2.5rem]`，绿色描边
- 标题：「🏆 毕业证书」bubbly-font 大号
- 显示：总星数、总积分、最终称号、日期
- 底部按钮：`[再玩一次]` `[炫耀一下]`

### 动效时序
```
t=0s    证书从下方弹入（spring 弹性动画）
t=0.3s  ⭐ 逐个放大弹出（stagger 错开）
t=0.6s  称号从透明滑入 + 粒子礼花爆发
t=0.9s  总分数字滚动递增（counter roll）
t=1.5s  底部按钮淡入
t=2s~   持续飘落彩色纸屑（Canvas 粒子复用）
```

### 称号等级
```
🌱 英语小芽     0 分
🌿 英语小苗   500 分
🌳 英语小树  1500 分
🔥 英语学霸  3000 分
👑 单词大王  5000 分
```
