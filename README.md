# Word Blast Pro

面向小学2年级（7-8岁）的中英单词学习 PWA 游戏。通过**配对消消乐 → 字母拼写 → 完形填空**三小节闭环，训练认读→默写→运用能力。

## 技术栈

| 类别 | 选型 |
|---|---|
| 框架 | React 19 + TypeScript 6 |
| 构建 | Vite 8 |
| 样式 | Tailwind CSS 4 |
| 渲染 | Canvas 2D（配对模式） |
| 音效 | Web Audio API + SpeechSynthesis（TTS） |
| PWA | vite-plugin-pwa（离线可用、可安装） |

## 快速开始

```bash
cd frontend
npm install
npm run dev      # 开发模式
npm run build    # 生产构建 → dist/
```

## 关卡结构

12关 × 3小节线性推进：

| 关卡 | 内容 | 小节 |
|---|---|---|
| 1-9 | 基础单词（90词） | 配对消消 → 字母拼写 → 完形填空 |
| 10-11 | 核心短语（12组） | 短语配对 → 选词拼短语 → 单词拼写 |
| 12 | 完整句子（6句） | 句子配对 → 完形填空 → 连词成句 |

## 积分 & 称号

- 配对/拼写/填空正确：+100
- Combo 加成：+50 × 连击数
- 求助机制：首字母提示(-300) → 揭露一半(-500) → 跳过(-800)

| 称号 | 分数 |
|---|---|
| 🌱 英语小芽 | 0 |
| 🌿 英语小苗 | 500 |
| 🌳 英语小树 | 1500 |
| 🔥 英语学霸 | 3000 |
| 👑 单词大王 | 5000 |
