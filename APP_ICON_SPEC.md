# App 图标设计规格

## 项目背景

Word Blast Pro — 面向 7-8 岁儿童的中英单词学习 PWA 游戏。配对消消乐 + 字母拼写 + 完形填空，游戏化英语启蒙。

## 图标用途

PWA 可安装应用图标，出现在手机/平板主屏、桌面快捷方式。

## 尺寸要求

请一次性生成以下 5 个尺寸的方形 PNG：

| 尺寸 | 文件名 | 用途 |
|---|---|---|
| 192×192 | `icon-192.png` | Android 主屏 |
| 512×512 | `icon-512.png` | PWA manifest、启动页 |
| 180×180 | `icon-180.png` | iPhone 主屏 |
| 152×152 | `icon-152.png` | iPad 主屏 |
| 1024×1024 | `icon-1024.png` | 高清兜底 / App Store |

## 设计规范

- **形状：** 纯方形，不要圆角、不要透明底（系统会自动裁切圆角）
- **背景：** 不透明，建议亮色
- **风格：** 扁平化、卡通趣味、儿童友好
- **可辨识度：** 缩小到 48px 仍能看清主体

## 配色建议

| 颜色 | 色值 | 用途 |
|---|---|---|
| 柠檬绿 | `#84cc16` | 主背景 / 品牌色 |
| 青柠渐变 | `#fef08a` → `#d9f99d` | 柔和渐变背景 |
| 亮白 | `#ffffff` | 主体图形 |
| 深绿 | `#166534` | 点缀 / 文字（可选） |

## 设计方向（三选一）

### 方案 A：字母 + 星爆
- 主体：一个大写字母 **W** 和 **B** 交错叠加
- 周围：放射状小星星 / 火花粒子
- 底部：细微的绿色到黄色渐变背景

### 方案 B：单词卡片
- 主体：一张圆角卡片造型，上方英文 "cat"，下方中文 "猫"
- 卡片有轻微 3D 阴影
- 背景：柠檬绿纯色

### 方案 C：萌趣角色
- 主体：一个可爱的卡通书本或铅笔角色
- 书本翻开，飞出彩色字母粒子
- 表情：😊 笑脸
- 配色：绿色系为主，点缀暖黄

### 推荐
**方案 A** — 最简洁，缩小后辨识度最高，适合 App 图标场景。

## 参考风格

- 扁平化设计（Flat Design）
- 类似 Duolingo、Khan Academy Kids 的图标风格
- 不要拟物化纹理，不要渐变金属反光
- 线条清晰、色彩明快
